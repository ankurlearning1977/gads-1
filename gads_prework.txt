===========
pre-work
===========

Please complete these steps before the course begins:

    step 1 (Mac users):
        install homebrew (a package manager for osx) --> http://brew.sh/

    step 1 (Linux users):
        familiarize yourself with the package manager native to your distribution; eg yum, apt-get, etc.

    Please come to the first class with a package manager installed & a basic idea of how to use it. We will install python and some relevant libraries together.

Please complete these steps by the end of week 1:

    step 2: unix review
    http://cli.learncodethehardway.org/book/

    step 3: python review
    https://developers.google.com/edu/python/
    http://learnpythonthehardway.org/

    optional: try some of these out
    http://projecteuler.net/
